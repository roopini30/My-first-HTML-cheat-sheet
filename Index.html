Basic HTML tags and attributes cheatsheet

HTML tags & its purposes

Tag Purpose
<!--...--> Defines a comment.
<button> Defines a clickable button.
<form> Defines a HTML form for user input.
<i> Defines a part of text in an alternative voice or mood.
<span> Defines a small section of a text or part of a document and also easily used for css styling.

HTML attribute,Components of and Purposes

Attribute Component of Purpose
action <form> specifies where to send the form-data when the form is submitted.
alt <area>,<img>,<input> specifies an alternate text when the original element fails to display.
cols <textarea> specifies the visible width of a text area.
colspan <td>,<th> specifies a number of columns a table call should span.
for <label>,<output> specifies which form element(s) a label/calculation is bound to.
